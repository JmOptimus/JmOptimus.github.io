Introduction

Imagine an alien form of life whose DNA strand is formed of two types of nucleobases (0/1) spreading to infinity.
Its evolution over time is ruled by a convolution along the strand using a 1xN kernel.
In other words, a sliding window of length N, traverse the infinite strand transforming a group of N nucleobases into one nucleobase according to a set of transformation rules.

The initial state

The begining of everything

Every dynamic system needs some initial conditions in order to start evolving;
our primordial seed consists of an infinite sequence of binary terms.
Given an initial state, a kernel size and a set of transformation rules , the system will evolve in a deterministic manner.

The cellular automata described by Wolfram, consists of an initial state with a central value of '1' preceded by infinite 0s to the left and followed by infinite 0s to the right.
The kernel dimensions for this particular case are 1x3.

Transformation rules

"to exist is to change"

Our kernel (sliding window for dummies), for a given moment in time, in the current generation, will be focusing on N contiguous nucleobases(lets call them cells from now on);
those cells will collaborate in the formation of one cell from the next generation.

In other words: $$f: \{0,1\}^{N} \mapsto \{0,1\} $$

As we can see, for a given kernel of length N, there are 2N different combinations; for each of those combinations, the resulting cell can take two possible outcomes (0/1).
The total number of rule sets that can be built is: $$2^{(2^{N})}$$

In our exploration adventure, our goal is to fill a lattice of cells considering the current exploration parameters (displacement and depth).

For a given total depth D, a kernel length Kl, lattice dimensions lw X lh, the total cells that need to be stored is

The initial state (S0) will have a length $$L_{0}=(K_{l}-1)*D+l_{w}$$

$$\frac{K_{l}-1}{2}D^{2} + \left(\frac{K_{l}-1}{2}+l_{w}\right)D + l_{w}$$ $$, D=d+l_{h}$$
From the equation above, is observed a quadratic space complexity $$\mathcal{O}(D^{2})$$ as a function of the exploration depth.
